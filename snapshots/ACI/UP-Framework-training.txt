CONTROL CENTER INFRAESTRUCTURE

Is a Service-Oriented Architecture (SOA)
Defines a core set of services and capabilities
Capabilites invoked from the COntrol Center application of from APIS exposed through the control center
Has a structure composed of a set of componets: (web presentation modules, business logic modules, data access modules)
Provides support for MFA(multifactor auth)


CONTROL CENTER MODULES
-Control center shell application: host the login page and present the desktop after success authentication
-Administration modules
-Entilementes,  security, authentication, autorization, roles and user management
-Business Intelligence: use UTR(universal Transaction Repository) and presents data in for of Dashboards
-Microservices INFRAESTRUCTURE: OpenAPIS,(REST) APIS 
-Plugings: The control Center modules contain the infa for addtional plug-ins
-Plaform Manger Capabilites: exposed in control center ar only applcable for Cloud deployments.
